Git Commands and Concepts

Generating SSH Key Pair

Command: ssh-keygen -t rsa -C "<emailid>"

After generating the SSH key pair, add the public key to your GitHub account in settings.

Configuring User Information

Commands:

  • git config --global user.name "username"
  • git config --global user.email "emailid"

Checking Remote Repositories

Command: git remote -v

Cloning a Repository

Command: git clone "<copy paste the ssh related url>"

Staging and Committing Changes

Commands:

  • git add .
  • git commit -m "message"

Checking Status and Logs

Commands:

  • git status
  • git log

Pushing and Pulling Changes

Commands:

  • git push
  • git pull

GUI Tools

Commands:

  • git gui&
  • gitk&

Version Tree

Commands:

  • git show -s --pretty=raw <SHA-1 hash>
  • git ls-tree <tree sha-1 hash>
  • git show <blob sha-1 hash>

Undoing Changes

Commands:

  • git checkout --<file>
  • git checkout .
  • git clean -xdf
  • git reset -- <file.txt>
  • git reset HEAD~n
  • git commit --amend -m "commit message"
  • git reset --soft HEAD~1
  • git reset --mixed HEAD~1
  • git reset --hard HEAD~1
  • git revert <sha1>

Git Ignore

Create a .gitignore file in the repository and write the names of the files which should be ignored.

Examples:

  • *.log to ignore all .log files
  • !error.log to track error.log file even though you are ignoring .log files
  • /TODO to only ignore the TODO file
  • build/ to ignore all files in the build/ directory
  • doc/*.txt to ignore all .txt files only in the doc directory but not in subdirectories
  • doc/**/*.pdf to ignore all .pdf files in the doc directory and in subdirectories

Branching and Merging

Commands:

  • git checkout -b <branchname> to create a new branch
  • git checkout <branchname> to switch to that branch
  • git merge <branchname> to merge the branch into the current branch

Tips to Avoid Conflicts:

  • Short commits
  • No edits to whitespaces
  • Merge often

Commands to Resolve Conflicts:

  • git merge --abort to abort merge or cancel the merge
  • git checkout --Xours or --Xtheirs to resolve by selecting version
  • git diff to resolve merge manually
  • git revert <sha1> to undo merge

Rebase and Cherry Pick Commands

Tagging Commits

Commands:

  • git tag <tagname>
  • git tag --list to see all the tags
  • git push --tags to push tags separately
  • git checkout <tagname> to navigate to that particular tag

Stashing Changes

Commands:

  • git stash save "<description>"
  • git stash list to view stashes
  • git stash pop to bring the changes back and remove from the stash
  • git stash apply to bring them back and leave in the stash
  • git stash drop to remove or clear them

Remote Repositories

Commands:

  • git remote -v to check which server we are working with
  • git remote remove origin to remove the existing remote
  • git remote add <name> <url> to add a remote

Extras

Commands:

  • git config --global core.editor <path>
  • git blame
  • git bisect
  • git log --pretty=oneline to see all commits in one line
  • git log --pretty=format:"%h %s" -graph
  • git config --global alias.last
  • git last
  • git log master..experiment
  • .git rerere
  • git submodule
  • git filter-branch --tree-filter
  • git tag -d tag-name where tag-name is the name of the Git tag you want to delete
  • git tag to list all tags
  • git push --all origin to push all branches

Keyboard Shortcuts

Shortcuts:

  • Ctrl + Insert for copying
  • Shift + Insert for pasting
